# Match-app Strategic Context

## Vision
フリーランス・コンサルタント・経営者が事業・スキル・案件を通じて最適なパートナーを見つけるプラットフォーム。AIが双方のニーズを理解し、質の高いマッチングを提案する。

## Target Users
1. **フリーランス**: 案件獲得、スキルマッチング
2. **コンサルタント**: クライアント開拓、専門領域マッチング
3. **経営者・事業主**: パートナー探し、外注先発見、事業提携

## Core Value Proposition
- 🎯 AIマッチング: プロフィール・実績・ニーズからAIが最適候補を推薦
- 🌊 簡単UI: 誰でも3分で登録、直感的な操作
- 🤝 複合マッチング: 1つのプラットフォームで案件・人材・提携を横断検索

## Key Differentiators
- AI推薦モード（マッチ度スコアリング + 理由説明）
- マルチロール対応（1アカウントで複数の立場で利用可能）
- 事業マッチング特化（求人サイトではなく事業パートナーシップ）

## Technical Decisions
- Server Components優先（SEO + 初期表示速度）
- PostgreSQL + Drizzle ORM（型安全、マイグレーション管理）
- Better Auth（認証 — OAuth + メール）
- AIマッチング: 将来的にGemini/Claude API連携

## Risks & Mitigations
- コールドスタート問題 → 初期はマニュアルキュレーション
- マッチング精度 → ユーザーフィードバックループで改善
- 収益モデル → フリーミアム + プレミアムマッチング
